产线自动测试平台
-------------------------------------------------------------------------------

平台：
Python 2.7 Windows
PyQt GPL 4.8.3
pyserial-2.7.win32
python-msp430-tools

-------------------------------------------------------------------------------
版本演进记录：

1.00.00		
	初版

1.00.01	
	1、改进用例自动触发机制；
	2、增加异步服务端业务引擎，上传测试报告改为异步动作；
	3、演进了GS10的测试逻辑。

1.00.02
	1、增加命令行参数，能够支持自定义配置文件及窗口位置；
	2、修正输入框，支持随时回车提交及默认值；
	3、加入了GS15的唐山曹妃甸卡的测试项。
	4、将GS15的测试支持为单机测试，增加保存本地记录文件的引擎
	5、单机登录区分用户，使用super登录为超级用户，其他或不写用户名为普通用户，不支持参数设置及测试项选择
	6、增加测试结果的醒目显示，用于自动触发的情况
	
1.00.03
	1、增加本地工具模块，能够类似测试用例集和测试资源，以插件的形式部署本地工具
	2、增加了本地数据库支持引擎
	3、增加兰州电子车牌的测试及发卡用例，增加了兰州发货数据导出本地工具
	4、增加了公告栏功能，能够始终驻留界面进行公告提示

1.10.00
	整理上述改动，发布次版本号：1.10.00
	支持的产品：GS10、GS11、GS15（南京环保卡、曹妃甸、兰州）及安智博通用工装；
	增加测试资源支撑：Alien9900、Zxris8801、Zxris6700、NuLink、Windows标准摄像头、ZBar条码扫描
				龙岗16I/16O控制板、SHHIC GS25发卡器、大华镭雕机
				标准USB扫码枪、KEYENCE扫码枪、民德FS580AT扫码枪
				GS10工装板
	1、增加从服务端发布的参数（SP工具）的本地设置。
	
2.00.00
	20160705开始开发2.00版本，主要改动
	1、调整架构，增加对批量测试的支持。能够支持串行/并行两种方式的批量测试。
	2、从华虹剥离，理应发布新的版本 ：）


2.01.00
    201806开发，主要改动：
    1、增加上传测试报告到服务器的断链缓存
    2、增加服务参数的本地化支持
    3、本地测试绑定数据、结果数据的保存于查询
